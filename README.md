## ✨ 核心模块

- **🌸 思想花园 (Blog)**：以杂志排版风格展示的技术笔记与随笔，支持平滑的阅读体验与深度视觉适配。
- **🚀 项目陈列馆 (Projects)**：个人作品集展示，包含动态遮罩、技术栈标签及详尽的项目背景说明。
- **🧪 任务实验室 (Plan)**：基于 Kanban 看板的任务追踪系统，支持拖拽交互与任务状态语义化管理。
- **💭 思维显影板 (ChatNotes)**：对话式的思考片段记录，采用“黑板/白板”双风格切换与关键词云动态生成。
- **📚 资源库 (Resources)**：精心筛选的开发工具、设计灵感与学习路径导航。

## 🎨 视觉特色

- **🌓 深度双色模式**：针对亮色与暗色模式进行了像素级的对比度校准，确保在任何光线下都有极致的阅读舒适度。
- **✨ 动态粒子背景**：基于 Canvas 开发的交互式粒子系统，随鼠标移动实时反馈，并与全局主题色同步。
- **🪟 现代磨砂质感**：广泛应用 Glassmorphism (毛玻璃) 效果，配合自适应边框流光动效，打造轻盈的层级感。
- **🎬 流畅交互动画**：全站使用 Framer Motion 实现物理级丝滑过渡，增强用户的交互沉浸感。

## 🛠️ 技术栈

- **框架**: [React 19](https://react.dev/)
- **构建工具**: [Vite 7](https://vitejs.dev/)
- **语言**: [TypeScript](https://www.typescriptlang.org/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **动画**: [Framer Motion](https://www.framer.com/motion/)
- **图标**: [Lucide React](https://lucide.dev/)
- **主题管理**: [next-themes](https://github.com/pacocoursey/next-themes)

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone <your-repository-url>
cd Kimi_Agent_数字花园视觉升级/app
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 构建生产版本
```bash
npm run build
```
